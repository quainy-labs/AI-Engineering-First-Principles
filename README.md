# AI Engineering from First Principles

**Build Reliable Intelligent Systems That Improve Real Work**

This repository contains the working manuscript and planning artifacts for Quainy's AI Engineering book.

The book is a builder's field manual, not an AI encyclopedia. It teaches one durable capability:

> Take a messy real-world workflow, decide whether AI is useful, design a reliable intelligent system, build it, evaluate it, deploy it responsibly, and improve it with feedback.

## Quainy Principle

The book is built around proof of capability:

- baseline;
- system boundary;
- information design;
- model interface;
- evaluation;
- observability;
- risk review;
- deployment plan;
- public build note.
