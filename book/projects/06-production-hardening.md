# Project 6: Production Hardening

## Goal

Turn prototype into system people can trust.

## Output

Create:

```text
projects/production-hardening/
  evaluation-plan.md
  observability-and-cost-plan.md
  risk-review.md
  deployment-and-improvement-plan.md
```

## Required Controls

System must have:

- eval dataset;
- release gate;
- tracing/logging;
- cost estimate;
- latency budget;
- access control;
- prompt injection tests;
- rollback plan;
- feedback loop.

## Evaluation

Measure:

- regression pass rate;
- production failure rate;
- cost per task;
- p95 latency;
- user acceptance;
- security test results.

## Final Artifact

Ship production readiness review.
