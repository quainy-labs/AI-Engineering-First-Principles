## Capstone

[Build One Real Intelligent System](capstone.md)

Build one real intelligent system from a real workflow. Deliver:

- problem brief;
- workflow map;
- baseline;
- architecture;
- working prototype;
- evaluation dataset;
- error analysis;
- risk review;
- cost and latency estimate;
- demo;
- public build note.
